# Galeria de Memórias — Pastoral da Caridade

Galeria web estática com carrossel 3D interativo e cilindro fotográfico em segundo plano.

## Estrutura

```
galeria-memorias/
├── index.html          # Página principal
├── css/
│   └── main.css        # Estilos
├── js/
│   ├── data.js         # Fotos, títulos e legendas
│   └── gallery.js      # Lógica da galeria
├── assets/
│   ├── 01.png … 10.png
└── README.md
```

## Como visualizar

Abra `index.html` no navegador ou execute `run.bat` (Windows).

## Personalização

Edite `js/data.js` para alterar títulos, legendas ou nomes dos arquivos em `assets/`.

## Integração em outro site

Copie a pasta inteira para o servidor ou repositório e aponte um link para `index.html`.
Mantenha a estrutura de pastas (`css/`, `js/`, `assets/`).

## Requisitos

- Navegador moderno (Chrome, Edge, Firefox, Safari)
- Conexão apenas para carregar as fontes Google (opcional: hospedar fontes localmente)
